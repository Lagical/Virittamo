# Virittamo
MERN Stack Application for Recording Issued Devices
 
## Getting started

### Install node packages
`npm install`

### Run frontend
`npm run dev`

### Run backend
`node server.js`

### Configuring MongoDB
create .env file to root folder and add following line to the file:  
`REACT_APP_MONGODB_URI=<YOUR_MONGODB_URI_HERE>`
